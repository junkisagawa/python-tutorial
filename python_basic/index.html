    Python基本文法

    1. 変数と型

    なぜ学ぶのか？

    プログラムは「データを操作する」もの。データの種類（型）や変数の使い方を知ることで、思い通りに情報を扱えるようになります。

    現場での使い所

    • ユーザー入力やファイルの内容を変数に保存
    • 計算やデータ変換、APIレスポンスの加工など

    解説

    Pythonは「動的型付け」なので、型を意識しすぎず直感的に書けます。

    num = 10        # 整数
pi = 3.14       # 小数
name = "AI"     # 文字列
is_ok = True    # 真偽値

    型変換

    age = "20"
age = int(age)  # 文字列→整数

    2. 条件分岐

    なぜ学ぶのか？

    「条件によって動きを変える」ことは、どんな業務システムでも必須です。

    現場での使い所

    • 入力値のチェック
    • 権限による画面切り替え

    解説

    score = 75
if score >= 80:
    print("合格")
else:
    print("不合格")

    3. ループ（繰り返し）

    なぜ学ぶのか？

    「大量のデータを一括処理」するのがプログラムの強みです。

    現場での使い所

    • ファイルの全行を処理
    • APIから取得した複数データの一括変換

    解説

    for i in range(5):
    print(i)

    4. 関数

    なぜ学ぶのか？

    「同じ処理を何度も書かない」ための仕組みです。

    現場での使い所

    • データ変換やバリデーションの共通化
    • 複雑な処理の分割

    解説

    def greet(name):
    return f"こんにちは、{name}さん"

print(greet("山田"))

    5. リスト・辞書・セット

    なぜ学ぶのか？

    「複数のデータをまとめて管理」できると、業務効率が大幅アップ！

    現場での使い所

    • 顧客リスト、商品一覧、設定値の管理
    • APIレスポンスのパース

    解説

    fruits = ["りんご", "バナナ", "みかん"]
user = {"name": "田中", "age": 30}
unique_ids = {1, 2, 3}

    6. ファイル操作（実践例）

    なぜ学ぶのか？

    「データの保存・読み込み」は現場で必須。

    解説

    with open("sample.txt", "w", encoding="utf-8") as f:
    f.write("こんにちは\nPython!\n")

with open("sample.txt", "r", encoding="utf-8") as f:
    for line in f:
        print(line.strip())

    ループ（繰り返し）

    処理を繰り返し行います。

    # for ループ
for i in range(5):  # 0, 1, 2, 3, 4
    print(i)

# リストでのループ
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)

# 辞書でのループ
person = {"name": "Alice", "age": 25}
for key in person:
    print(f"{key}: {person[key]}")

# key, valueのループ
for key, value in person.items():
    print(f"{key}: {value}")

# while ループ
count = 0
while count < 5:
    print(count)
    count += 1

# break と continue
for i in range(10):
    if i == 3:
        continue  # このイテレーションをスキップ
    if i == 7:
        break  # ループを抜ける
    print(i)

    関数

    再利用可能なコードのブロックです。

    # 基本的な関数
def greet(name):
    """挨拶を返す関数"""
    return f"Hello, {name}!"

print(greet("Alice"))  # "Hello, Alice!"

# デフォルト引数
def greet_with_time(name, time="morning"):
    return f"Good {time}, {name}!"

print(greet_with_time("Bob"))  # "Good morning, Bob!"
print(greet_with_time("Carol", "evening"))  # "Good evening, Carol!"

# 可変長引数
def sum_all(*numbers):
    total = 0
    for num in numbers:
        total += num
    return total

print(sum_all(1, 2, 3, 4, 5))  # 15

# キーワード引数
def build_profile(first_name, last_name, **user_info):
    profile = {
        'first_name': first_name,
        'last_name': last_name
    }
    profile.update(user_info)
    return profile

user = build_profile('Albert', 'Einstein',
                    location='Princeton',
                    field='Physics')
print(user)

    リストと辞書

    複数のデータを効率的に管理します。

    リスト

    # リストの作成
fruits = ["apple", "banana", "cherry"]

# 要素へのアクセス
print(fruits[0])  # "apple"

# リストの操作
fruits.append("orange")  # 追加
fruits.insert(1, "blueberry")  # 挿入
fruits.remove("banana")  # 削除
popped = fruits.pop()  # 最後の要素を取り出し
print(popped)  # "orange"

# リストのスライス
numbers = [0, 1, 2, 3, 4, 5]
print(numbers[1:4])  # [1, 2, 3]
print(numbers[:3])   # [0, 1, 2]
print(numbers[3:])   # [3, 4, 5]
print(numbers[::2])  # [0, 2, 4] - 2ステップごと

# リスト内包表記
squares = [x**2 for x in range(10)]
print(squares)  # [0, 1, 4, 9, 16, 25, 36, 49, 64, 81]

even_squares = [x**2 for x in range(10) if x % 2 == 0]
print(even_squares)  # [0, 4, 16, 36, 64]

    タプル（不変リスト）

    # タプルの作成
dimensions = (200, 50, 100)

# 要素へのアクセス
print(dimensions[0])  # 200

# タプルは変更不可
# dimensions[0] = 250  # エラー

    辞書

    # 辞書の作成
person = {"name": "Alice", "age": 25, "city": "Tokyo"}

# 要素へのアクセス
print(person["name"])  # "Alice"

# 辞書の操作
person["email"] = "alice@example.com"  # 追加
person["age"] = 26  # 更新
del person["city"]  # 削除

# キーの存在確認
if "name" in person:
    print("Name exists!")

# get メソッド（キーがない場合のデフォルト値を指定）
phone = person.get("phone", "未登録")
print(phone)  # "未登録"

# 辞書内包表記
squares_dict = {x: x**2 for x in range(5)}
print(squares_dict)  # {0: 0, 1: 1, 2: 4, 3: 9, 4: 16}

    セット（重複なしコレクション）

    # セットの作成
fruits_set = {"apple", "banana", "cherry", "apple"}
print(fruits_set)  # 重複は自動的に除去される

# セットの操作
fruits_set.add("orange")
fruits_set.remove("banana")

# セットの演算
set_a = {1, 2, 3, 4, 5}
set_b = {4, 5, 6, 7, 8}
print(set_a.union(set_b))        # 和集合: {1, 2, 3, 4, 5, 6, 7, 8}
print(set_a.intersection(set_b))  # 積集合: {4, 5}
print(set_a.difference(set_b))   # 差集合: {1, 2, 3}

    エラー処理

    # 基本的な例外処理
try:
    number = int(input("数字を入力してください: "))
    result = 10 / number
    print(f"結果: {result}")
except ValueError:
    print("有効な数字を入力してください")
except ZeroDivisionError:
    print("0で割ることはできません")
except Exception as e:
    print(f"予期せぬエラー: {e}")
else:
    print("エラーなく処理が完了しました")
finally:
    print("処理を終了します")

    for line in file:
        print(line.strip())  # 末尾の改行を除去
