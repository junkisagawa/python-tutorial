# 付録

## 便利なツール・ライブラリ

- **開発環境**: VS Code, PyCharm
- **パッケージ管理**: pip, pipenv, poetry
- **テスト**: pytest, unittest, coverage
- **Web開発**: Flask, Django, FastAPI
- **データ処理**: pandas, numpy
- **可視化**: matplotlib, seaborn
- **API開発**: Flask-RESTful, FastAPI, flasgger
- **DB操作**: SQLAlchemy, SQLite, PostgreSQL
